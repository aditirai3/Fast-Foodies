About This Project

Using a dataset of ~10K fast food restaurants, we isolated five of the most prominent chains. We analyzed the dataset using the dashboard of visualizations below to identify the various densities: locations per state by chain, locations per million people with respect to poverty rates by chain, and locations by zip code with respect to population. Dataset used provided by Datafiniti via Kaggle.
